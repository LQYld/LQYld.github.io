# asyncData 方法

  • 类型： Function

你可能想要在服务器端获取并渲染数据。Nuxt.js 添加了 asyncData 方法使得你能够在渲染组件之前异步获取数据。

解析（asyncData 与 Data 的区别）

asyncData 会在渲染组件之前提前进行异步获取数据，然后将数据与 Data 中的进行合并。

asyncData 方法会在组件（限于页面组件）每次加载之前被调用。它可以在服务端或路由更新之前被调用。在这个方法被调用的时候，第一个参数被设定为当前页面的上下文对象，你可以利用 asyncData 方法来获取数据并返回给当前组件。

export default {
  data() {
    return { project: "default" };
  },
  // return 出去的对象格式跟 data 中的一样。
  asyncData(context) {
    return { project: "nuxt" };
  },
};

注意

由于 asyncData 方法是在组件 初始化 前被调用的，所以在方法内是没有办法通过 this 来引用组件的实例对象。

# asyncData 使用 Promise

解析（axios）

axios 是一个基于 ajax 封装的第三方请求库。

以使用 axios 请求为例。

export default {
  asyncData() {
    return axios.get("https://xxx.xxxx.com/xxx/xxx").then((res) => {
      return { xxx: res.data };
    });
  },
};

# asyncData 使用 async & await

解析（async & await）

学过 ES6+ 的小伙伴们应该会有些了解，async & await 是 ES7 的新提案，基于 Promise 的语法糖，可以将异步转为同步。

export default {
  async asyncData() {
    let { data } = await axios.get("https://xxx.xxxx.com/xxx/xxx");
    return { xxx: data };
  },
};
